# 📊 Forecast Financial Model

This repository contains a **Forecast Financial Model** built using historical data. The model includes **assumptions, revenue forecasts, cost structures, and profit estimations** over a 12-month period.

## 📌 Overview

The financial model provides insights into:

- **Traffic Analysis**: Organic and paid page traffic trends.
- **Conversion Rates**: Monthly conversion rate variations.
- **Revenue Projections**: Estimated income from orders.
- **Cost Assumptions**:
  - **Variable Costs**: Manufacturing, fulfillment, and paid traffic costs.
  - **Fixed Costs**: Rent, salaries, and other administrative expenses.
- **Profitability Analysis**:
  - **Gross Profit & Margin** calculations.
  - **Contribution Margin** and percentage over time.
  - **EBT (Earnings Before Tax)** and net profit trends.

## 📊 Key Financial Metrics

| Metric          | Description |
|----------------|-------------|
| **Revenue**    | Projected monthly earnings based on traffic and conversions. |
| **COGS**       | Cost of Goods Sold, including production and fulfillment expenses. |
| **Gross Profit** | Revenue minus COGS, showing profitability before operating costs. |
| **Variable Costs** | Marketing and fulfillment expenses that scale with sales. |
| **Fixed Costs** | Operational expenses like rent and salaries. |
| **EBT (Earnings Before Tax)** | Net income before tax deductions. |
| **Net Profit** | Final profit after tax deductions. |

## 📈 Assumptions

- **Traffic Growth**: Organic and paid traffic increases at different rates.
- **Conversion Rate Variability**: Conversion rates fluctuate based on source.
- **Cost Changes**: Manufacturing and order fulfillment costs shift mid-year.
- **Fixed Expenses Stability**: Salaries, rent, and administrative costs remain consistent.
